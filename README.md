# 📒 Kas Vintage — Aplikasi Catatan Keuangan

Aplikasi web sederhana bergaya **vintage (buku kas klasik)** untuk mencatat pemasukan dan pengeluaran secara interaktif, ringan, dan tanpa backend.

## ✨ Fitur Utama

### 💰 Manajemen Keuangan

* Tambah transaksi (pemasukan & pengeluaran)
* Hapus transaksi
* Kategori custom (bisa tambah & hapus)
* Catatan tambahan pada transaksi

### 📊 Dashboard & Ringkasan

* Total saldo otomatis
* Total pemasukan & pengeluaran
* Jumlah transaksi

### 📜 Riwayat Transaksi

* Filter berdasarkan:

  * Hari ini
  * Minggu ini
  * Bulan ini
  * Tahun ini
* Filter berdasarkan jenis:

  * Pemasukan
  * Pengeluaran
* Fitur pencarian (search)

### 📈 Laporan Keuangan

* Grafik:

  * Bar chart (pemasukan vs pengeluaran)
  * Pie chart (distribusi kategori)
* Statistik:

  * Total pemasukan
  * Total pengeluaran
  * Selisih saldo
* Breakdown per kategori

### ⚙️ Pengaturan

* Kelola kategori (emoji + nama)
* Informasi aplikasi:

  * Total transaksi
  * Tanggal transaksi pertama
* Reset semua data (danger zone)

### 💾 Penyimpanan

* Menggunakan **IndexedDB**
* Data tersimpan permanen di browser (offline-ready)

---

## 🎨 Tampilan UI

* Desain **vintage klasik** seperti buku kas
* Warna dominan:

  * Coklat, emas, dan krem

## 🚀 Cara Menjalankan

1. Download / clone repository ini
2. Buka file:

```bash
kasir-vintage-v1.html
```

3. Jalankan di browser:

   * Google Chrome (disarankan)
   * Microsoft Edge
   * Firefox

---

## 🧪 Cara Penggunaan

### ➕ Menambah Transaksi

1. Pilih jenis:

   * Pengeluaran / Pemasukan
2. Isi:

   * Keterangan
   * Jumlah
   * Tanggal
   * Kategori
3. Klik **Simpan Transaksi**

---

### 🔍 Melihat Riwayat

* Gunakan filter waktu & jenis
* Gunakan search untuk mencari transaksi

---

### 📊 Melihat Laporan

* Pilih periode:

  * Bulanan / Tahunan / Semua
* Grafik akan otomatis diperbarui

---

### ⚙️ Mengatur Kategori

* Tambahkan kategori baru (emoji + nama)
* Hapus kategori jika tidak diperlukan
